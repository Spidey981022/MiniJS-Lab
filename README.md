# MiniJS-Lab

A collection of **100 JavaScript mini projects** built while practicing DOM manipulation and core JS concepts.  
Each project is designed to be small, focused, and practical — perfect for strengthening problem-solving skills and building a solid foundation in front-end development.

## 🚀 What’s Inside
- Interactive DOM-based projects (buttons, forms, lists, animations, etc.)
- Clean, well-commented code for learning and reference
- Organized folder structure with individual README files
- Test cases and examples for selected projects
- Continuous updates as new projects are completed

## 🎯 Goals
- Master DOM manipulation and JavaScript fundamentals
- Build a professional GitHub portfolio with consistent contributions
- Share learnings and progress with the developer community

## 📚 Inspiration
Projects inspired by [100 JS Projects](https://www.100jsprojects.com/) and extended with personal improvements, documentation, and testing.
